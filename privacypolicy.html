Privacy Policy

Last updated: February 20, 2025

Your privacy is important to us. This Privacy Policy explains how Litescan ("we", "our", or "the bot") collects, uses, and protects information when you use our Discord bot and website.

1. Data We Collect

When you invite Litescan to your Discord server or use the /scan command, we may collect:

  • User IDs & Server IDs: Necessary to identify members and servers for scanning.
  • Membership Information: Public data about which servers a user is in (only accessible via Discord API with proper permissions).
  • NSFW Server Database: We maintain a database of server IDs that have been flagged as NSFW. This database is built from public reports and submissions.
  • Usage Data: Aggregated, non-personal data about command usage to improve the bot.

2. How We Use Data

We use the collected data to:

  • Provide NSFW server detection and scanning functionality.
  • Improve and update our NSFW server database.
  • Respond to user support requests.
  • Ensure the security and integrity of the bot.

3. Data Sharing

We do not sell or rent your personal data. Limited data may be shared:

  • With Discord Inc. as necessary for API functionality.
  • If required by law or to protect our legal rights.

4. Data Retention

User IDs and server IDs are retained only as long as necessary to provide the service. You may request deletion of your data by contacting us via our support server.

5. Security

We implement reasonable security measures to protect your information. However, no method of transmission over the internet is 100% secure.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Continued use of the bot constitutes acceptance of the revised policy.
